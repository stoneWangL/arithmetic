题目描述
大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。
n<=39














思路：
public class Solution {
    public int Fibonacci(int n) {
        return func(n);
    }
    private int func(int m){
        if(m==0)
            return 0;
        else if(m==1 || m==2)
            return 1;
        else
            return func(m-1)+func(m-2);
    }
}
