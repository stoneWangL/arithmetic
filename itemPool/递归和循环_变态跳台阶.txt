题目描述
一只青蛙一次可以跳上1级台阶，也可以跳上2级……它也可以跳上n级。
求该青蛙跳上一个n级的台阶总共有多少种跳法。














思路：
public class Solution {
    public int JumpFloorII(int target) {
        return Func(target);
    }
    private int Func(int n){
        if(n==1)
            return 1;
        else
            return 2*Func(n-1);
    }
}
